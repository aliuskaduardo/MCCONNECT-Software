MCeCoNet

Markov Centralities for Complex Networks


MCeCoNet is a Python/wxPython application for centrality study of any complex network and performs two types of calculations:
  • Markov topological indices (TIs) during a network attack by consecutively removing the network nodes (the network is losing a node during each attack step);
  • Markov centralities for each removed node (the network is losing only one node).


How to use MCeCoNet

In the main windows you can choose the calculation parameters and the input/output files:
  • Files: network (*.mat), results, attack TIs amd centralities files;
  • Markov Centralities: select the indice to be calculated and the power;
  • Node attack:
    • the type of attack (classical connectivity based on the node degrees, average probability for each node,  entropy of each node, node asymmetries as difference between  the averaged probability by line and column, autocorrelation as the product of the same averaged probabilities and by keeping the original matrix without no sorting);
    • the attack steps and if will be created the centrality graphs;
    • selection of  the centrality/TI line plots and visualize them (with gnuplot);
  • Network: generate and show the initial and final networks (with Graphviz);
All the options have default values in order to perform minimum of calculations. The processing of the network can be seen in a console window. If you will close it, all the application windows will close too.

If MCeCoNet will raise any error, you can see it in the same console and please send us in order to fix it. 


Markov Centralities (MC)

Before to calculate the TIs and the correspondent centralities, the connectivity matrix is Markov normalized (the matrix element is divided by the maximum in its matrix row) resulting a matrix of node probabilities (P). In the second step, these P matrix is raised to the power k, resulting k matrices (Pk), the input for all the TI calculations. The node Markov centralities are calculated as the difference between the TIs of the original matrix and the TIs without the correspondent node.
